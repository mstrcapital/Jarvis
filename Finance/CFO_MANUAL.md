```python
class TreasuryManager:
    """国库管理器"""
    
    def __init__(self):
        self.balance = 0
        self.reserves = 0
        self.income = 0
        self.expenses = 0
        
    def check_balance(self):
        """检查余额"""
        # 从ClawRouter获取
        pass
    
    def record_income(self, source, amount):
        """记录收入"""
        self.income += amount
        self.balance += amount
        self._log_transaction("INCOME", source, amount)
    
    def record_expense(self, category, amount):
        """记录支出"""
        if amount > self.get_approval_threshold():
            raise Exception("需要审批")
        self.expenses += amount
        self.balance -= amount
        self._log_transaction("EXPENSE", category, amount)
    
    def transfer_to_reserves(self, amount):
        """转入储备金"""
        if amount > self.balance:
            raise Exception("余额不足")
        self.balance -= amount
        self.reserves += amount
    
    def get_financial_report(self):
        """生成财务报告"""
        return {
            "balance": self.balance,
            "reserves": self.reserves,
            "income": self.income,
            "expenses": self.expenses,
            "net_flow": self.income - self.expenses
        }
```

## 🎓 财务知识

### 关键术语

| 术语 | 说明 |
|------|------|
| 现金流 | 现金流入和流出 |
| 收支平衡 | 收入=支出 |
| 储备金 | 应急资金 |
| ROI | 投资回报率 |
| 预算 | 计划支出 |

### 财务管理原则

1. **量入为出**: 不超支
2. **留有余地**: 保持储备
3. **记录清晰**: 每笔都记
4. **定期复盘**: 每周/每月
5. **分散风险**: 不过度依赖单一收入
