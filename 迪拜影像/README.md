## 使用方法

1. **连接Instagram**：
   - 登录Postiz
   - 添加Instagram Channel
   - 授权

2. **配置工具**：
   - 编辑 `工具/postiz-upload.js`
   - 填入API Key

3. **运行**：
   ```bash
   cd 工具
   node postiz-upload.js
   ```
