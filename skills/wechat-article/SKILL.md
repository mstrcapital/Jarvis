## 选题策略

### 选题来源

| 来源 | 优先级 | 更新频率 |
|------|--------|----------|
| Trump Truth Social | ⭐⭐⭐ | 每日3次 |
| X (Twitter) 财经账号 | ⭐⭐⭐ | 每日4次 |
| 路透/彭博突发 | ⭐⭐⭐ | 实时监控 |
| 政策/立法动态 | ⭐⭐ | 每日1次 |
| 社群/读者反馈 | ⭐⭐ | 按需 |

### 选题公式（核心）

**参考Larry的爆款公式:**
> **[具体人群] + [痛点/担忧] + [价值信息] + [行动引导]**

**Examples:**
- "美股投资者最怕的3件事，今天全发生了"
- "Trump昨晚这条动态，90%的人没看懂"
- "开盘前5分钟，这个信号必须关注"

### 选题类型

| 类型 | 占比 | 说明 |
|------|------|------|
| 每日简报 | 40% | 固定栏目，建立期待 |
| 热点解读 | 35% | 追踪Trump/美股热点 |
| 深度分析 | 20% | 独家观点，长文输出 |
| 互动/测试 | 5% | 提高粉丝粘性 |

### 选题检查清单

在发布前检查:
- [ ] 是否有独特视角？（不是复读新闻）
- [ ] 是否与读者利益相关？
- [ ] 是否有时效性？
- [ ] 是否有争议/讨论空间？
- [ ] 我有什么独特观点？

## 内容结构

### 标准文章结构（6段式）

```
1. 开头Hook（50字内）
   - 抛出问题/悬念
   - 引发好奇心

2. 背景铺垫（100-150字）
   - 简述事件背景
   - 建立上下文

3. 核心内容（500-800字）
   - 重点信息解读
   - 数据/事实支撑
   - 逻辑清晰

4. 分析预判（200-300字）
   - 我的观点
   - 可能的影响
   - 未来展望

5. 行动建议（100-150字）
   - 读者应该怎么做
   - 或引发思考

6. 结尾互动（50-100字）
   - 提问/投票
   - 引导留言
```

### 每篇必备元素

| 元素 | 位置 | 说明 |
|------|------|------|
| 日期标注 | 开头 | "2026年2月15日" |
| 分区标识 | 文中 | "Trump动态" / "美股速递" |
| 数据来源 | 引用处 | 增强可信度 |
| 个人观点 | 分析区 | 标明"我认为是..." |
| 互动问题 | 结尾 | 引导留言 |

### 禁止事项

- ❌ 纯新闻搬运（要有解读）
- ❌ 过时信息（发布前检查时效性）
- ❌ 过于绝对的语言（"一定"、"肯定"）
- ❌ 未证实的小道消息（除非标注）
- ❌ 投资建议（只能说"个人看法，不构成建议"）

### 每日简报生成Prompt

```
你是一个专业的财经自媒体编辑，负责撰写微信公众号文章。

## 角色
- 名字：Jarvis
- 风格：专业、有观点、不端水
- 擅长：美股分析、Trump新闻解读

## 任务
根据以下素材，撰写一篇公众号文章。

## 素材
{素材内容}

## 要求
1. 标题：14-30字，简洁有力，引发好奇
2. 摘要：30-50字，概括核心观点
3. 正文：800-1200字
4. 结构：遵循6段式（Hook→背景→核心→分析→建议→互动）
5. 语言风格：
   - 专业但不晦涩
   - 有个人观点输出
   - 敢于表达态度
   - 避免端水式废话
6. 配图建议：在文中标注需要配图的位置

## 输出格式
- 标题：[标题]
- 摘要：[摘要]
- 正文：[完整文章]
- 配图建议：[列出需要配图的位置和内容]

## 注意事项
- 不是新闻搬运，要有解读和观点
- 引用数据时标注来源
- 涉及投资时加粗提示"个人看法，不构成投资建议"
```

## 成功公式

### 高表现内容特征

**高打开率标题:**
- 数字具体："3个信号" / "90%的人"
- 悬念设置："发生了什么" / "没看懂"
- 利益相关："你必须知道" / "关乎你的钱"
- 对比反差："不是...而是..."

**高转发内容:**
- 有独特观点（值得分享）
- 有信息差（别人不知道）
- 有实用价值（收藏转发）
- 有讨论空间（引发话题）

**高互动内容:**
- 结尾有提问
- 话题有争议
- 邀请表态
