# Cancel Block Trade Order (TRADE)

## API Description[​](https://developers.binance.com/docs/derivatives/options-trading/market-maker-block-trade/Cancel-Block-Trade-Order)

Cancel a block trade order.

## HTTP Request[​](https://developers.binance.com/docs/derivatives/options-trading/market-maker-block-trade/Cancel-Block-Trade-Order)

DELETE `eapi/v1/block/order/create`

## Request Weight[​](https://developers.binance.com/docs/derivatives/options-trading/market-maker-block-trade/Cancel-Block-Trade-Order)

**5**
