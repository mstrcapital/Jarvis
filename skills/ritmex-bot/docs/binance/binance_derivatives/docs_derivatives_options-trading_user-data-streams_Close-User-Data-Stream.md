# Close User Data Stream (USER\_STREAM)

## API Description[​](https://developers.binance.com/docs/derivatives/options-trading/user-data-streams/Close-User-Data-Stream)

Close out a user data stream.

## HTTP Request[​](https://developers.binance.com/docs/derivatives/options-trading/user-data-streams/Close-User-Data-Stream)

DELETE `/eapi/v1/listenKey`

## Request Weight[​](https://developers.binance.com/docs/derivatives/options-trading/user-data-streams/Close-User-Data-Stream)

**1**

## Request Parameters[​](https://developers.binance.com/docs/derivatives/options-trading/user-data-streams/Close-User-Data-Stream)

None

## Response Example[​](https://developers.binance.com/docs/derivatives/options-trading/user-data-streams/Close-User-Data-Stream)

```
{}
```
