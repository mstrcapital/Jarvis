## Tools

### x-reader (内容抓取)

Use x-reader to fetch content from various platforms for research and content creation:

```bash
# Fetch content from any URL
x-reader <url>

# Examples:
x-reader https://mp.weixin.qq.com/s/abc123    # 微信公众号
x-reader https://www.xiaohongshu.com/explore/abc123  # 小红书
x-reader https://www.bilibili.com/video/BVxxx  # B站
x-reader https://x.com/elonmusk/status/123456  # Twitter
x-reader https://youtube.com/watch?v=xxx       # YouTube
x-reader https://RSS_FEED_URL                  # RSS
```

### Content Sources

| Platform | Command | Use Case |
|----------|---------|----------|
| 微信公众号 | `x-reader <url>` | 抓取文章 |
