## 📞 常见问题

### Q: 如何添加新素材？

A: 将文件放到对应素材库的 `素材/` 目录，更新 `素材库归档.md`

### Q: 如何生成新样本？

A: 复制 `样本/sample-post-001-v2.md`，修改后保存到 `样本/` 目录

### Q: 视频如何发布？

A: 目前需要手动下载后，用 `postiz-upload.js` 上传到Postiz

### Q: 三个素材库可以共用内容吗？

A: 可以，但建议针对平台特性调整文案和配图
