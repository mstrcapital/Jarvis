## 📋 接单流程

### Step 1: 浏览任务

```bash
每日任务:
- 早晨: 浏览ClawRouter新任务
- 午间: 检查新增任务
- 傍晚: 筛选合适任务
```

### Step 2: 评估任务

| 评估项 | 问题 | 权重 |
|--------|------|------|
| 能力匹配 | 我能做吗？ | 高 |
| 时间预估 | 需要多久？ | 高 |
| 收益分析 | 值得做吗？ | 高 |
| 客户评价 | 信誉如何？ | 中 |
| 风险评估 | 有坑吗？ | 高 |

### Step 3: 竞标/接单

```python
评估模型:

def evaluate_task(task):
    score = 0
    
    # 能力匹配 (30分)
    if task.match_our_skills():
        score += 30
    
    # 时间合理 (25分)
    if task.time_estimate < task.deadline:
        score += 25
    
    # 收益达标 (25分)
    if task.reward >= min_reward:
        score += 25
    
    # 风险低 (20分)
    if task.risk_low():
        score += 20
    
    return score >= 80  # 80分以上才接
```

### Step 4: 执行任务

```bash
标准流程:
1. 确认需求
2. 制定计划
3. 执行开发
4. 交付成果
5. 客户验收
6. 收款确认
```

### Step 5: 复盘优化

```markdown
复盘清单:
- [ ] 任务是否按期完成？

- [ ] 有哪些可复用成果？
- [ ] 效率能否提升？
- [ ] 下次如何做得更好？
```
