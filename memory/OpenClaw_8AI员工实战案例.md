## 🔧 二、多模型配置

### 使用aigocode.com中转

```yaml
# Claude (不带/v1)
{
  "api": "anthropic-messages",
  "baseUrl": "https://api.aigocode.com"
}

# GPT/Codex (带/v1)
{
  "api": "openai-responses",
  "baseUrl": "https://api.aigocode.com/v1"
}
```

### ⚠️ 关键坑点

```
Claude: 不带 /v1
OpenAI: 必须带 /v1

搞反必报错！
```

### 模型分工

| 模型 | 用途 | 成本 |
|------|------|------|
| Claude | 深度思考、复杂任务 | 付费 |
| Codex | 写代码、定时任务 | 免费 |

### 故障转移 (Watchdog)

```
每5分钟检测:
├─ 主模型工作 → 继续使用
└─ 主模型挂了 → 自动切备用模型
```
