## ⚠️ 注意事项

### 法律合规

```
✅ 可以做的:
├── 生成原创内容
├── 销售自己的作品
├── 制作教程
└── 合规使用公共素材

❌ 不能做的:
├── 下载有版权的水印视频
├── 二次销售他人作品
├── 未经授权使用明星脸
└── 生成虚假新闻
```

### 平台规则

```
闲鱼:
- 禁止盗版素材
- 需要原创证明
- 遵守价格规则

小红书:
- 原创内容优先
- 禁止引流到其他平台
- 遵守广告法

B站:
- 禁止搬运
- 需要版权证明
- 遵守社区规范
```
