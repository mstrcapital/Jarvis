## 交易纪律清单

### 1. 仓位管理 (Position Sizing)
- [ ] 单笔交易最大亏损：___% 账户
- [ ] 单日最大亏损：___% 账户
- [ ] 亏损50%后需要100%回本——严守风控底线

### 2. 市场选择 (Table Selection)
- [ ] 优先选择"软桌子"：低流动性市场、预测市场
- [ ] 避免直接进入 FX/指数等高效市场
- [ ] 评估市场竞争程度再做决策

### 3. 置信度缩放 (Conviction Scaling)
- [ ] 强信号（高置信度）：___% 仓位
- [ ] 中等信号：___% 仓位
- [ ] 弱信号：___% 仓位或观望
- [ ] 像算牌者一样根据 count 调整

### 4. 过程 > 结果 (Process Over Outcome)
- [ ] 每次交易后问："我是否遵守了系统？"
- [ ] 区分"决策质量"与"运气"
- [ ] 正确决定也可能亏钱——不因短期结果否定过程
- [ ] 记录每笔交易的决策理由，而非仅记录结果

### 5. 情绪韧性 (Emotional Resilience)
- [ ] 自动化执行交易，排除情绪干扰
- [ ] 避免报复交易
- [ ] "禅宗机器人"思维——只关注下一个决策
- [ ] 不受连败/连赢影响决策
