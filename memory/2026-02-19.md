### AI发布公众号实战 (@sitinme)
- **Source**: https://twitter.com/sitinme/status/2023930738773815314
- **效率**: 2小时 → 10分钟 (节省90%)
- **流程**: access_token → 上传图片 → HTML生成 → 推送到草稿箱
- **最大坑**: 中文乱码 (\uXXXX转义)
  - 解决: `json.dumps(data, ensure_ascii=False).encode("utf-8")`
- **其他坑**:
  - 标题限制20个中文字 (45003错误)
  - 图片必须上传到微信素材库
  - HTML用 p + inline style
  - IP白名单是代理出口IP
- **搭建**: `npm install -g openclaw && openclaw onboard` + aigocode.com API Key
