## 🎯 交易策略

### 市场选择

**重点关注**: 天气预测市场 (Polymarket)

选择标准:
1. 高机会分数 (>30)
2. 外部价格 vs 市场价格的差异 (>10%)
3. 即将到期 (<24h)
4. 流动性充足 (>1000 volume)

### 限价挂单

**注意**: Simmer平台不支持限价挂单，仅Polymarket支持

对于真实交易:
```json
{
  "venue": "polymarket",
  "order_type": "GTC",
  "price": 0.08
```

### 交易规则

1. **每笔交易**: 最大 $10 (模拟) / $100 (真实)
2. **每日限额**: $500
3. **止损**: 50%
4. **止盈**: 35%

### 交易思路

1. **研究市场**: 检查分辨率标准、当前价格
2. **检查上下文**: 获取仓位和警告
3. **制定论点**: 为什么要买这个结果
4. **执行交易**: 包含推理过程
