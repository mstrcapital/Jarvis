## 📋 收集方案

由于无法直接下载版权图片，请按以下步骤操作：

### 方案1: 手动收集 (推荐)

#### 步骤1: 访问Hermes官网

```
1. 打开浏览器
2. 访问: https://www.hermes.com/
3. 搜索 "Birkin" 或 "Kelly"
4. 右键图片 → 另存为
```

#### 步骤2: 保存图片

**保存位置**:
```

├── Birkin/
│   └── Birkin_30/
│       └── Black_Togo/
│           └── [Hermes官网图片].jpg
│

        └── Black_Epsom/

```

