## 核心功能

50+ API聚合:
- Twitter/X: 用户查询、推文、粉丝
- Web搜索: Google、新闻、图片
- 区块链: 代币价格、钱包分析
- AI服务: 文本生成、图像分析

## 快速开始

注册:
curl -X POST https://api.xapi.to/api/auth/register

保存凭证到 ~/.config/xapi/credentials.json
